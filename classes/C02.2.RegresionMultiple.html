Coeficientes de determinacin

Es el valor \(R^2\) de la regresin. Es la proporcin de la varianza de la variables de respuesta que puede ser predicha a partir de las variables explicativas.

El valor de \(R^2\) vara entre 0 y 1. Cuanto mayor es, menos incerteza se tiene de los valores reales a partir del modelo. Es decir, mejor es la prediccin.

\[ \text{sum of squares of the data} = SS_{data} = \sum_i{(y_i-\bar{y})^2} \\ \text{sum of squares of prediction} = SS_{pred} = \sum_i{(fit(y_i)-\bar{y})^2} \\ R^2 = \frac{SS_{pred}}{SS_{data}} \]

# Calculo la media de los valores observados
mean_y <- mean(Y)

# Calculo la suma de cuadrados de las observaciones
ss_data <- sum((Y-mean_y)^2)

# Calculo la suma de cuadrados de las predicciones
ss_pred <- sum((my_model$fitted.values-mean_y)^2)

# Calculo el coeficiente de determinacin
R2a <- ss_pred/ss_data
R2a
## [1] 0.9985323
# Extraigo el coeficiente de determinacin del modelo.
R2b <- summary(my_model)$r.squared
R2b
## [1] 0.9985323

Veamos como cambia el \(R^2\) con el error del modelo.

nvars <- 2
nvalues <- 50
variables <- replicate(nvars, {runif(nvalues)})

alpha <- runif(1)
alpha
## [1] 0.4053067
betas <- runif(nvars)
betas
## [1] 0.8175160 0.5202013
par(mfrow=c(2,2))
for (i in c(0.05, 0.2, 0.5, 1)) {
  errors <- i * runif(nvalues)
  Y <- alpha + variables %*% betas + errors
  my_model <- lm(Y ~ variables)
  plot(
    Y ~ my_model$fitted.values,
    main=paste("R2", round(summary(my_model)$r.squared, 4)),
    pch=19,
    col="blue"
  )
}

Supuestos

  • Linealidad: Si no se cumple se comete un error de especificacin, dado que el modelo ser incorrecto. En el caso de una o dos variables independientes, ayuda observar un grfico de dispersin en busca de una tendencia lineal (lnea, plano).

  • Independencia de los residuos: Los residuos no deben estar correlacionados (deben ser una variable aleatoria). El estadstico de Durbin-Watson puede ayudar en la deteccin de autocorrelaciones.

  • Homocedasticidad: Igualdad de varianza para cada valor o grupo de valores de la/s variable/s independiente/s

  • Normalidad: Para cada valor o grupo de valores de la/s variable/s independiente/s los residuos se distribuyen de manera normal con \(\mu = 0\)

  • No-colinealidad: Las variables independientes no pueden ser combinaciones lineales unas de otras, no debe haber correlacin entre ellas.

Mnimos Cuadrados Ponderados.

En el caso de no cumplirse con la homocedacia, es posible darle a cada observacin un peso inversamente proporcional a la varianza para ese valor o grupo de valores de la/s variable/s independiente/s en una regresion por minimos cuadrados ponderados.

Lo ptimo es conocer de antemano cual es la varianza de cada medicin. Por ejemplo si medimos cada dato con una herramienta diferente y conocemos su error. Sino existen maneras de determinar los posibles valores de la varianza (o de un estimador robusto correlacionado con sta) con respecto a las variables independientes.

Vamos a modelar los residuos. Primero vamos a estimar la varianza localmente usando bins.

my_bins <- as.data.frame(cbind(
  start=seq(0,0.9,0.05),
  midpoint=seq(0,0.9,0.05)+0.05,
  end=seq(0,0.9,0.05)+0.1
))

mad_bins <- apply(
  my_bins,
  MARGIN = 1,
  FUN = function(x) {
    r <- residuals(my_model)
    r <- r[x["start"]<values & values<=x["end"]]
    mad(r)
  }
)

my_bins <- cbind(
  my_bins,
  mad=mad_bins
)
plot(

Buscando colinearidad

Calculemos las correlaciones.

La correlacin y la covarianza son medidas de variacin conjunta entre dos variables aleatorias.

La covarianza es una medida que depende de las unidades y escalas de las variables aleatorias, por lo que no siempre es til. La correlacin es una medida sin unidades que vara entre -1 y 1. Lo cual la hace mucho ms fcil de trabajar e interpretar.

\[ cov(X,Y) =\frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{n-1} \\ cor(X,Y) = cov(X,Y)/(\sigma_x \sigma_y) \]

a <- c(1,3,5)
b <- c(3,4,7)
cov(a,b)
## [1] 4
ad <- a - mean(a)
bd <- b - mean(b)
cv <- sum(ad*bd) / 2

## [1] 0.9607689

Calculamos el Variance Increasing Factor

El VIF es una medida de la colinearidad en un sistema de regresin multiple. Es una medida del incremento de la varianza en el modelo completo vs al modelo con una sola varible.

Se calcular un valor de VIF para cada variable aleatoria del modelo.

# Calculo un modelo donde la variable a evaluar
# Year en este caso es respuesta de las dems
# variables aleatorias del modelo.

my_vif_model <- lm(

  data=longley[-1]

# Calculo el coeficiente de determinacin
r2 <- summary(my_vif_model)$r.squared
