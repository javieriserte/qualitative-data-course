Estadstica inferencial

La estadstica inferencial trata de inferir informacin acerca de la distribucin de probabilidad de una poblacin, partiendo de la informacin de una muestra. Se diferencia de la estadstica descriptiva en que en lugar de simplemente describir la muestra, trata de describir algo acerca de la poblacin. Uno puede realizar diferentes estimaciones acerca de la poblacin, la cual normalmente se conduce haciendo uso de la estadstica frecuentista:

  • Estimacin puntual: Estimacin de un parmetro poblacional haciendo uso del valor que mejor lo aproxime.
  • Estimacin por intervalos: Estimacin de un intervalo de valores para un parmetro poblacional desconocido. Por ejemplo, la estimacin de un intervalo de confianza para un parmetro poblacional.
  • Test de hiptesis

Test de hiptesis

Una prueba de hiptesis es anloga a una demostracin matemtica por contradiccin (en particular es un tipo de reductio ad absurdum), aunque no debe considerarse en ningn caso una demostracin matemtica. Ambos mtodos, parten de considerar una declaracin verdadera, en el caso del test de hiptesis, la hiptesis nula H. En ambos mtodos se trata de llegar a una contradiccin, dado que una implicacin solo puede ser falsa si partiendo de una premisa verdadera llegamos a una conclusin falsa (tabla de verdad):

p q p q
true true true
true false false
false true true
false false true

Ac es donde ambos mtodos divergen, dado que la demostracin matemtica lo hace siguiendo reglas lgicas determinadas y procedimientos matemticos permitidos, llegando a un resultado determinado no azaroso. Finalmente, si se llega a una contradiccin, se concluye que la declaracin original es falsa, aceptando la negacin de dicha declaracin. En el caso de la prueba de hiptesis, rechazar H implica aceptar la hiptesis alternativa H.

Sin embargo, no rechazar H no implica aceptarla, decimos simplemente que no hay evidencia suficiente para rechazar la hiptesis nula. En cualquier caso, una prueba de hiptesis no demuestra una hiptesis, slo la soporta con un cierto grado de confianza. Acumular evidencia a favor de una hiptesis (razonamiento inductivo) no es suficiente para demostrarla, la evidencia acumulada solo puede soportar dicha hiptesis.

Existen dos tipos de errores que podemos cometer (Neyman & Pearson, 1933):

H es cierta H es cierta
Se escogi H Error de tipo I Decisin correcta
Se escogi H Decisin correcta Error de tipo II

El riesgo o probabilidad de caer en un error del tipo I se conoce como nivel de significacin estadistica, normalmente indicado con la letra griega \(\alpha\). La probabilidad de caer en un error del tipo II normalmente se anota con la letra griega \(\beta\). Siendo \(1-\beta\) la potencia o poder del test.

\[P(escoger H_a | H_0 cierta) = \alpha\] \[P(escoger H_0 | H_a cierta) = \beta\]

\[P(escoger H_a | H_a cierta) = 1 - \beta\]

Valor P

El valor P (p value) es la probabilidad de observar un determinado valor (o uno ms extremo) dado que H es correcta. En ningn caso debe interpretarse como una probabilidad de la hiptesis nula. Un valor P bajo indica que los datos son inconsistentes con la hiptesis nula, permitindonos rechazar la hiptesis nula ante esa observacin. Para decidir que es un valor bajo, uno puede recurrir al valor de significacin .

\[P(escoger H_a | H_0 cierta) = P(p \le \alpha | H_0 cierta) = \alpha\]

T de Student para una muestra

La prueba de T de Student para una muestra permite probar si la media de una muestra es diferente de una media \(\mu_0\). Algo que tenemos que tener en cuenta, es que no podemos testear que la media poblacional sea un determinado valor. sto se debe a que no podemos saber cul es la media poblacional sin analizar toda la poblacin. Es decir, no podemos saber dnde se encuentra localizada la distribucin T Student correspondiente a la media poblacional. Para determinar probabilidades necesitamos un parmetro de ubicacin exacto. Es por esto que la hiptesis nula ser la ubicacin de la media poblacional en un lugar determinado. De hecho, las hiptesis nulas se caracterizan normalmente por tener igualdades (uno supone una distribucin con parmetros determinados).

\[H_0 : \mu = \mu_0\]

Si para una muestra de tamao \(n\) se cumplen las supuestos del test:

  • Distribucin normal de la variable (continua) en la poblacin
  • Muestreo al azar (cada observacin es independiente)

Entonces el estadstico \(t\) sigue una distribucin T de Student con \(n - 1\) grados de libertad.

\[ t = \frac{ \bar{x} - \mu }{ \frac{s}{\sqrt{n}} }\]

Supongamos una variable aleatoria \(X\) que en la poblacin se distribuye con una distribucin Normal(0.1,0.1). Y supongamos que estamos interesados en demostrar que la media de esa variable \(X\) en la poblacin es mayor a 0. Es decir:

\[H_0 : \mu = 0\]

\[H_a : \mu > 0\]

Otras dos hiptesis alternativas son posibles; \(H_a \ne 0\) \(H_a < 0\). Podemos elegir cualquiera de stas dependiendo de nuestro hiptesis de trabajo, determinando si el test ser a una o dos colas. En este caso el test ser a una cola y el p value para un valor observado \(T\) ser:

\[P(t \ge T|H_0)\]

n <- 5

s <- sd(normal_sample)

# Hipotesis Nula. La distribucin es T de Student

T_value <- (xm-m0)/(s/sqrt(n))

El p value depende de si el test es a una o dos colas, en este caso, como la distribucin es simtrica alrededor del 0, es obvio que el p value para el test a dos colas es el doble del p value que obtuvimos. Pero calcular el p value a dos colas no es trivial, principalmente puede ser un problema para distribuciones asimtricas. Pero en general se toma la idea de doblar el valor del P value menor al testear las dos opciones a una cola (es como si hiciramos una correccin de Bonferroni al haber realizado dos pruebas).

En particular, para una variable aleatoria \(X\), que sigue una determinada distribucin si \(H_0\) es cierta, dado un estimador \(x_0\), el p value es:

\(P(X \geq x|H_0)\) prueba de una cola hacia la derecha y \(P(X \leq x|H_0)\) prueba de una cola hacia la izquierda.

Intervalo de confianza

El intervalo de confianza es un tipo de estimador por intervalo para un parmetro de la poblacin. Se calcula a partir de una o ms muestras de la poblacin, y posee un nivel de confianza que determina qu tan frecuentemente una serie de intervalos estimados de la misma manera contiene al parmetro poblacional.

Es importante notar que el intervalo de confianza calculado a partir de una nica muestra puede no contener al parmetro a estimar, pero an as es una buena estimacin intervalar del mismo. As un intervalo con una confianza del 95%, significa que de un nmero N de intervalos calculados, 0.95N contendrn al parmetro a estimar. No debe interpretarse nunca como un intervalo con una probabilidad del 95% de contener el parmetro.

El intervalo para una media se calcula utilizando el valor crtico de \(t\) si la desviacin estndar proviene de la muestra o \(Z\) si se conoce la desviacin estndar de la poblacin. Los valores crticos son aquellos valores de \(t\) o \(Z\) que dejan un rea de \(\frac{\alpha}{2}\) hacia la izquierda del menor valor crtico, y el mismo hacia la derecha (dejando un rea \(1-\alpha\) en el intervalo).

As el valor crtico, utilizando la distribucin T de Student con \(n-1\) grados de libertad, para una confianza del 95% (\(\alpha = 5%\)) puede calcularse con la funcin qt (para calcular cuantiles).

t_left <- qt(0.05/2, df = n-1) # 2.5 %
t_right <- qt(1-0.05/2, df = n-1) # 97.5 %

t_left
## [1] -2.776445
t_right
## [1] 2.776445

Los valores son iguales en mdulo dada la simetra de la distribucin T de Student. Siendo \(t\) ese mdulo, el intervalo de confianza para la media de la poblacin sera:

\[(\bar{x}-t\frac{s}{\sqrt{n}} , \bar{x}+t\frac{s}{\sqrt{n}})\]

c(
    mean(normal_sample + t_left*sd(normal_sample)/sqrt(n)),
    mean(normal_sample + t_right*sd(normal_sample)/sqrt(n))
)
## [1] -0.12614059  0.05861384

Bootstrap

Bootstrap nos permite obtener estimaciones puntuales o por intervalos, en particular, intervalos de confianza, para parmetros poblacionales sin necesidad de conocer la distribucin subyacente.

Bsicamente la tcnica de bootstrap se basa en hacer un muestreo aleatorio con reemplazo de nuestra muestra, calculando el parmetro de inters en cada muestra.

La muestra de la que se parte debe ser una muestra representativa, en particular debe complir con ser una muestra de valores independientes e idnticamente distribuidos. Si bien la distribucin de la variable de inters en la poblacin no forma parte de los supuestos del mtodo, una implementacin nave puede generar resultados incorrectos con distribuciones con colas pesadas. Bootstrap tambin puede presentar problemas con distribuciones discretas y poco densas.

my_sample <- c(
    rnorm(50, 3, 2),
    runif(50, 0,6)
)

replicates <- replicate(
    1000,
    sample(
        x = my_sample,
        size=100,
        replace=TRUE
    )
)

means <- apply(replicates, 2, mean)
hist(means, 100)

quantile(means, 0.025)
##     2.5% 
## 2.629036
quantile(means, 0.975)
##    97.5%

Prueba de bondad de ajuste a una distribucin.

Muchas veces es necesario saber cuando los datos de una muestra se ajustan a una determinada distribucin. En ese sentido uno se puede preguntar si la distribucin observada (en general, para una muestra) es igual, o no, a una distribucin esperada (en general, para la poblacin). Para testear sto, disponemos de las siguientes pruebas:

Prueba

La prueba se puede utilizar para testear la bondad de ajuste de cualquier distribucin univariada para la cual se pueda calcular la CDF. Dado que requiere datos en grupos, por lo que puede utilizarse para testear distribucin para datos categricos (a diferencia de KS). Para poder utilizar el test en una distribucin de probabilidad para datos continuos, debemos [discretizar/agrupar] los datos de la misma manera en que lo hacemos al calcular un histograma.

Debemos tener en cuenta q el test es sensible a la eleccin de los bins. Una desventaja de este test es que requiere de un tamao muestral de al menos 1000 para que la aproximacin sea vlida.

observed <- rbinom(1000, 100, 0.05)
hist(observed, 100)

xs <- seq(0, max(observed), 1) 
observed_cdf <- ecdf(observed)
plot(
    x=xs,
    y=observed_cdf(xs),
    pch=19,
    col="red")
points(
    x=xs,
    pbinom(q=xs, size = 100, p=0.055)
)

Dadas \(k\) categoras, para la categora \(i\), \(O_i\) representa la frecuencia observada para esa categora, mientras \(E_i\) representa la frecuencia esperada (nuestra hiptesis nula).

\[ ^{2} = \sum_{i=1}^{k}{\frac{(O_i - E_i)^{2}}{E_i}} \]

sta ltima, en el caso de haber generado bins para una variable continua, la calculamos usando la CDF evaluada en el extremo superior bin, menos su valor en el extremo inferior del bin. N es el tamao de la muestra.

\[Ei=N(CDF(X_f)CDF(X_i))\]

Oi <- 1000 * observed_cdf(xs)
Ei <- 1000 * pbinom(q=xs, size = 100, p=0.054)

chi2 <- sum((Oi - Ei)^2/Ei)

df = 15 # (16 bins - 1) * (2 cond - 1)

pchisq(
    chi2,
    df,
    lower.tail=FALSE
    )
## [1] 0.00408144

Prueba de Kolmogrov-Smirnov

La prueba de Kolmogorov-Smirnov es otra prueba de ajuste a una distribucin que usa la funcin de densidad acumulada (CDF).

En particular, su estadstico, \(D\) mide la distancia entre la CDF de la distribucin, que suponemos correcta en la hiptesis nula, y la ECDF de la muestra. En particular, el estadstico \(D\) es el valor mximo de las diferencias. Si la funcin de densidad acumulada emprica es igual a la funcin de densidad de nuestra distribucin a ajustar, el estadstico \(D\) ser 0 (\(H_0\): \(D = 0\)).

ste es un test paramtrico, por lo que hace pocas supuestos, pero debe tenerse en cuenta que la distribucin que suponemos en nuestra \(H_0\) no puede tener parmetros libres, debe estar completamente determinada y debe ser continua. Es sensible tanto a cambios en la ubicacin como en la forma de la distribucin, pero no tiene gran poder, por lo que puede requerir un tamao muestral grande para rechazar correctamente la hiptesis nula.

Test de signos

En muestras pareadas, si la \(H_0\) es verdad, el nmero de veces que la diferencia entre pares es positiva debe distribuirse como una binomial con probabilidad 0.5.

sample_1 <- rexp(25, 1)
sample_2 <- sample_1 + runif(25, -3, 3)

positives <- sum(sample_2>sample_1)
negatives <- sum(sample_2<sample_1)

pbinom(
    c(positives,negatives),
    size = 25,
    prob = 0.5,
    lower.tail = FALSE
)

Test de la mediana

Es un caso especial de la prueba de chi-cuadrado. Esta prueba posee poco poder.
Se calcula la mediana para todos los datos (\(x\) e \(y\)), establecindose dos grupos (por encima o por debajo de sta). Luego se arma una tabla de contingencia con esta nueva variable versus las dos variables anteriores (\(x\) e \(y\)).

sample_1 <- rbinom(25, 15, 0.04 )
sample_2 <- rbinom(25, 15, 0.1 )
m <- median(c(sample_1, sample_2))

table(c(sample_1, sample_2)>=m,
    rep(1:2, each=25))
##        
##          1  2
##   FALSE 11  5
##   TRUE  14 20
chisq.test(
    table(
        c(sample_1, sample_2)>=m,
        rep(1:2, each=25)))
## 
##  Pearson's Chi-squared test with Yates' continuity correction
## 
## data:  table(c(sample_1, sample_2) >= m, rep(1:2, each = 25))
## X-squared = 2.2978, df = 1, p-value = 0.1296

Test de Fischer

Se usa si se tienen dos variables nominales, y se quiere saber si las proporciones de una variable se mantiene entre los valores de la otra variable.

sample_1 <- rbinom(25, 15, 0.04 )
